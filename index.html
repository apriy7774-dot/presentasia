1. Apa itu C++?

C++ Adalah Bahasa pemrograman tingkat menengah yang sangat cepat dan efisien. bahasa ini juga digunakan untuk performa tinggi, untuk membuat program yang berat dan cepat, seperti Game, Sistem Operasi, dan Software Berat.

Contoh Script C++

#include <iostream>
int main() {
    std::cout << "Halo, ini C++";
    return 0;
}

Cara Pakai: Harus di-compile (diubah ke file .exe) baru bisa jalan.

Hasilnya: Muncul tulisan di terminal/layar hitam.

2. Apa itu JavaScript?

JavaScript: Adalah bahasa untuk membuat website jadi interaktif, seperti tombol yang bisa diklik atau animasi. biasanya Digunakan untuk Website interaktif, Aplikasi Web, dan Mobile Apps.

Contoh Script JavaSript

console.log("Halo, ini JavaSript");
alert("Selamat datang!");

Cara Pakai: Tinggal buka browser (Chrome), buka Console, langsung jalan.

Hasilnya: Muncul tulisan di console atau kotak pesan (pop-up).

3. Perbedaan Utama

C++: Harus di-compile, jalan di sistem, sangat cepat.

JavaScript: Langsung jalan di browser, interaktif, dan fleksibel.

Intinya: C++ itu buat performa (mesinnya),

JavaScript itu buat tampilan web (interaksinya)
